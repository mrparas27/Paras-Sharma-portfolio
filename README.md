# Paras-Sharma-portfolio

A world-class, futuristic 3D developer portfolio website built for Paras Sharma — AI/ML Engineer, Data Scientist, and Data Analyst.

## 🚀 Live Demo Server
*   Dev URL: `http://localhost:5173/`

## 🛡️ Technologies Used
*   **Core**: React (v19) & Vite
*   **3D Elements**: Three.js, React Three Fiber (R3F), and `@react-three/drei`
*   **Styling**: Tailwind CSS (v4) with Custom Glassmorphism Theme
*   **Animations**: Framer Motion (for viewport transitions) and Canvas-Confetti (for celebration triggers)
*   **Icons**: Lucide React

## 💎 Features
1.  **3D Interactive Background**: A slow-orbiting neon particle storm that shifts dynamically to mirror cursor movement.
2.  **3D AI Assistant Core**: An interactive holographic 3D Core Avatar following mouse look-at targets, integrated with a floating terminal chat box containing a local keyword matcher answering queries about education, experience, and projects.
3.  **3D Project Carousel**: Interactive 3D cards laid out in a cylindrical carousel that smoothly rotate and change focal lengths on user click or navigation arrows.
4.  **Bento About Me Grid**: Clean, Apple-inspired dashboard grid presenting education details (B.Tech CSE 2026) and industrial internship achievements from Maruti Suzuki.
5.  **3D Rotating Skills Tag Cloud**: A spherical 3D tag cloud hosting core skills (PyTorch, SQL, Pandas, NumPy, etc.) adjusting its velocity dynamically.
6.  **Holographic Certifications**: Certifications from Physicswallah and Qspiders that burst into confetti celebrations on click.
7.  **Transmitting Contact Terminal**: A contact portal structured as an encrypted packet transmitter with simulated transmission logs.

## 💻 Local Setup

1.  Clone the repository:
    ```bash
    git clone https://github.com/mrparas27/Paras-Sharma-portfolio.git
    cd Paras-Sharma-portfolio
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Compile a production build:
    ```bash
    npm run build
    ```
